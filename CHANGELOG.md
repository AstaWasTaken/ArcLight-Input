# Changelog

All notable changes to ArcLight Input will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.6] - 2025-12-15

### Changed
- Improved package distribution (includes dependencies)
- Updated documentation with comprehensive API reference
- Enhanced README with badges and better examples

### Fixed
- Package structure for Wally distribution

## [1.0.5] - 2025-12-15

### Changed
- Removed external dependencies from manifest (bundled internally)

## [1.0.4] - 2025-12-15

### Changed
- Updated package metadata
- Improved `.wallyignore` configuration

## [1.0.3] - 2025-12-15

### Added
- Initial Wally registry publication
- Complete Core modules implementation
- Type definitions

### Features
- Priority-based input management
- Input buffering system
- Sequence/combo detection
- Multi-device support (Keyboard, Mouse, Gamepad, Touch)
- UI integration helpers

## [1.0.0] - 2025-12-15

### Added
- Initial release
- Core input management system
- ActionRegistry module
- BufferSystem module
- DeviceDetector module
- InputProcessor module
- SequenceDetector module
- UIIntegration module
- Complete type definitions
